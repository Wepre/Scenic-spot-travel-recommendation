# 基于小程序云开发的旅游景点团建系统

# 基于小程序云开发的旅游团建小程序源码。该小程序前端采用小程序，后端用云开发，后台用的是cms内容管理系统。可以对景区进行浏览、对路线进行推荐、对用户进行注册、登录、下单、支付、评论等操作。

## 小程序端：
- 个人信息管理（头像、网名、手机号）
- 轮播图管理
- 景区展示，
- 用户登录注册
- 景区搜索，
- 景区具体信息浏览
- 景区推荐
- 路线推荐
- 路线推荐详情
- 跟团游方案管理
- 定制出行景区方案管理
- 下单管理
- 订单取消
- 订单支付
- 订单确认
- 订单评论

## 后台：

- 景区管理
- 路线方案管理
- 订单管理
- 用户管理
- 评论管理
